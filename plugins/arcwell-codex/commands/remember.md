# Remember

The user invoked this command with: $ARGUMENTS

Add the supplied fact to personal memory using `mem0_add` when it is clearly a
stable personal fact or preference. If the text is ambiguous, sensitive, or
extracted from a larger source, use `memory_capture` in review mode so it can be
reviewed before applying. Use `memory_extract_candidates` for lower-level
candidate inspection, and use `memory_add` only as the compatibility fallback.

Keep personal memory separate from wiki knowledge.
